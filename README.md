# Horse Farm Tours Website

Официальный сайт компании Horse Farm Tours, предлагающей экскурсии по конным фермам в Кентукки.

## Особенности

- Современный адаптивный дизайн
- Информация о турах и экскурсиях
- Галерея фотографий
- Система бронирования
- Отзывы клиентов
- Контактная информация

## Технологии

- HTML5
- CSS3
- Font Awesome (иконки)
- Адаптивный дизайн

## Установка и запуск

1. Клонируйте репозиторий:
```bash
git clone https://github.com/your-username/horse-farm-tours.git
```

2. Откройте `index.html` в браузере или запустите локальный сервер:
```bash
python -m http.server 8000
```

3. Откройте в браузере: http://localhost:8000

## Развертывание на GitHub Pages

1. Создайте новый репозиторий на GitHub
2. Загрузите файлы в репозиторий
3. В настройках репозитория включите GitHub Pages
4. Выберите ветку main как источник
5. Сайт будет доступен по адресу: https://your-username.github.io/horse-farm-tours/
